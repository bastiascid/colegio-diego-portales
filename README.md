# Colegio Diego Portales Machalí - Sitio Web Rediseñado

## 🎓 Descripción

Rediseño moderno y premium del sitio web del Colegio Diego Portales de Machalí. Este proyecto transforma el sitio original de Google Sites en una experiencia web contemporánea, manteniendo todo el contenido existente mientras mejora dramáticamente la estética y la experiencia del usuario.

## ✨ Características

- **Diseño Moderno**: Interfaz premium con gradientes, glassmorphism y animaciones suaves
- **Totalmente Responsivo**: Optimizado para dispositivos móviles, tablets y escritorio
- **Colores Institucionales**: Paleta de azul marino y dorado que refleja la identidad del colegio
- **Animaciones Interactivas**: Micro-interacciones y efectos hover para mejor engagement
- **SEO Optimizado**: Meta tags, estructura semántica y mejores prácticas
- **Navegación Intuitiva**: Menú claro y accesible en todas las páginas

## 📁 Estructura del Proyecto

```
colegio-diego-portales/
├── index.html              # Página principal
├── quienes-somos.html      # Información del colegio y equipo
├── comunicados.html        # Noticias y anuncios
├── reglamentos.html        # Documentos institucionales
├── guias-tareas.html       # Material educativo por nivel
├── css/
│   └── main.css           # Sistema de diseño completo
├── js/
│   └── main.js            # Funcionalidad interactiva
└── images/
    ├── logo.png           # Logo del colegio
    └── hero-bg.jpg        # Imagen de fondo hero
```

## 🚀 Cómo Usar

1. **Abrir el sitio**: Simplemente abre `index.html` en tu navegador
2. **Navegación**: Usa el menú superior para acceder a las diferentes secciones
3. **Responsive**: El sitio se adapta automáticamente a cualquier tamaño de pantalla

## 📱 Páginas Incluidas

1. **Inicio**: Hero section, visión institucional, sellos educativos y acceso rápido
2. **Quiénes Somos**: Presentación del colegio, instalaciones y equipo
3. **Comunicados**: Noticias y anuncios con diseño de tarjetas coloridas
4. **Reglamentos**: Documentos institucionales con enlaces de descarga
5. **Guías y Tareas**: Material educativo organizado por nivel (Pre-Kinder a 8º Básico)

## 📝 Próximos Pasos

Para implementar el sitio en producción:

1. **Actualizar Enlaces**: Reemplazar los enlaces "#" con URLs reales de Google Drive para documentos y guías
2. **Agregar Fotos**: Incluir fotos reales del personal docente y asistentes
3. **Configurar Hosting**: Subir a GitHub Pages, Netlify o servidor web
4. **Dominio**: Configurar dominio personalizado si es necesario
5. **Analytics**: Agregar Google Analytics para seguimiento
